## 🛠️ How to Install and Use 

> How to download and install :
- To start you should atleast have python and django installed.
- Clone this repo using : `git clone https://github.com/ursmaheshj/Student_Management.git`
- *Note*: make sure you are inside root folder before you run below commands
- Optional: create virtual environment using virtualenv or venv
- Install requirements with (`pip install -r requirements.txt`)
- Run collectstatic command (`python manage.py collectstatic`)
- Run makemigration command (`python manage.py makemigrations`)
- Run migrate command (`python manage.py migrate`)
- Run superuser command and provide email & password (`python manage.py createsuperuser`)
- Run server with (`python manage.py runserver`)

> How to use :
1. After logged in with superuser you can create teachers and students.
2. Once teachers and students profiles created they can login to their profiles using their mails and Default passwords(Teacher@100 and Student@100) which can be updated after login.
3. Teachers can create and manage student and also add results and notifications for students.
4. Students can login to their profiles and check their results and notifications posted by their teachers.
